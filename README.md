# Сегментация клиентов банка (Cluster Analysis)

## Описание проекта
Проект по кластерному анализу данных. Цель работы — сегментировать клиентов банка на основе их финансового поведения для выявления однородных групп.

В качестве датасета используется **Credit Card Dataset for Clustering**, содержащий информацию о тратах, лимитах, частоте операций и других финансовых показателях клиентов.

## Стек технологий
*   **Python 3**
*   **Pandas**: загрузка и предобработка данных
*   **NumPy**: численные вычисления
*   **Matplotlib / Seaborn**: визуализация данных
*   **Scikit-learn**: масштабирование признаков, кластеризация (K-Means, DBSCAN, Agglomerative Clustering), метрики качества

## Структура репозитория
*   `pis_pr2.ipynb` — Jupyter Notebook с полным кодом анализа: от загрузки данных до визуализации кластеров.
*   `README.md` — описание проекта.

## Этапы выполнения работы

1.  **Загрузка и осмотр данных**:
    *   Загрузка датасета.
    *   Проверка на наличие пропущенных значений (`NaN`) и их заполнение медианой.
    *   Удаление идентификатора клиента (`CUST_ID`, так как он не несет полезной нагрузки для кластеризации).

2.  **Предобработка данных (Preprocessing)**:
    *   Анализ распределения признаков (выявление асимметрии).
    *   Логарифмирование skewed-признаков (например, `BALANCE`, `PURCHASES`) для уменьшения асимметрии.
    *   Масштабирование признаков с помощью `StandardScaler`.

3.  **Анализ и визуализация**:
    *   Построение матрицы корреляций для отбора признаков.
    *   Применение метода **t-SNE** для визуализации многомерных данных в 2D пространстве.

4.  **Кластеризация**:
    *   Применение трех алгоритмов кластеризации:
        *   **K-Means**
        *   **DBSCAN**
        *   **Agglomerative Clustering** (Иерархическая кластеризация)
    *   Подбор гиперпараметров (`eps` для DBSCAN через K-distance graph, `k` для K-Means через elbow method/silhouette score).

5.  **Оценка качества**:
    *   Сравнение моделей с использованием метрик:
        *   Silhouette Score
        *   Davies-Bouldin Index
        *   Calinski-Harabasz Index

## Как запустить проект

1.  Установите необходимые библиотеки:
    ```bash
    pip install pandas numpy matplotlib seaborn scikit-learn
    ```

2.  Скачайте датасет (обычно доступен на Kaggle как "Credit Card Dataset for Clustering") и поместите его в папку с проектом, либо используйте путь, указанный в ноутбуке.

3.  Откройте файл `pis_pr2.ipynb` в Jupyter Notebook или Google Colab и последовательно запускайте ячейки.

## Результаты
В ходе работы были построены кластеры, характеризующие разные типы клиентов (например, "активные пользователи кредиток", "клиенты, снимающие наличные", "ответственные плательщики" и т.д.). Сравнение метрик позволило выбрать наиболее подходящий алгоритм кластеризации для данного набора данных.
